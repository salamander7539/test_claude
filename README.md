# React Time Series Chart

Интерактивный многолинейный график для отображения 4 временных рядов данных с различными типами визуализации.

## Возможности

- **4 типа визуализации**: Area (область), Spline (сглаженная линия), Line (прямая линия) и Bar (столбцы)
- **Интерактивная подсказка** с отображением всех метрик для выбранной даты
- **Адаптивный дизайн** с чистым, современным стилем
- **Настраиваемые данные** - легко подключить свои временные ряды
- Создано с использованием React и Recharts

## Быстрый старт

### Требования

- Node.js 16+ и npm

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/salamander7539/test_claude/

```

2. Установите зависимости:

3. Запустите сервер разработки:

4. Откройте браузер по адресу `http://localhost:5173`

## Использование

### Подключение своих данных

График принимает проп `data` с 4 временными рядами. Отредактируйте `src/App.jsx` для настройки:

```javascript
const yourData = {
  cost: [
    { x: new Date('2026-06-01').getTime(), y: 25.5 },
    { x: new Date('2026-06-03').getTime(), y: 30.2 },
    // ... дополнительные точки данных
  ],
  cpa: [
    { x: new Date('2026-06-01').getTime(), y: 2.1 },
    // ... дополнительные точки данных
  ],
  roi: [
    { x: new Date('2026-06-01').getTime(), y: 120.5 },
    // ... дополнительные точки данных
  ],
  conversions: [
    { x: new Date('2026-06-01').getTime(), y: 12 },
    // ... дополнительные точки данных
  ]
};

function App() {
  return (
    <div className="App">
      <TimeSeriesChart data={yourData} />
    </div>
  );
}
```

### Формат данных

Каждая последовательность — это массив объектов с полями:
- `x`: временная метка в миллисекундах (используйте `new Date('YYYY-MM-DD').getTime()`)
- `y`: числовое значение

### Типы визуализации

Компонент отображает 4 ряда со следующими типами:
1. **Cost (Стоимость)** - График области с градиентной заливкой (желтый)
2. **CPA** - Столбчатая диаграмма (синий)
3. **ROI confirmed (Подтвержденный ROI)** - Сглаженная линия (зеленый)
4. **Conversions (Конверсии)** - Линия с квадратными маркерами (фиолетовый)

## Настройка

Отредактируйте `src/components/TimeSeriesChart.jsx` для изменения:

- Высота графика
- Формат подсказки
- Стили линий и маркеров
- Максимальные значения осей
- Цвета фона
